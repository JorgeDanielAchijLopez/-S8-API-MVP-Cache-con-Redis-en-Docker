# InvControl Pro API

## Descripción

InvControl Pro es una API REST para la gestión de inventarios orientada a pequeños negocios. Permite registrar productos, gestionar ventas diarias y mantener actualizado el stock.

Como parte del MVP, se implementa un mecanismo de cache utilizando Redis para optimizar el rendimiento en consultas frecuentes de productos, reduciendo tiempos de respuesta y carga en la lógica principal.

El sistema está diseñado con una arquitectura simple basada en contenedores utilizando Docker y Docker Compose.

---

## Objetivo del proyecto

El objetivo de este proyecto es construir un MVP funcional que demuestre:

- Diseño e implementación de una API REST
- Uso de contenedores con Docker
- Integración de Redis como sistema de cache
- Documentación técnica clara y estructurada

---

## Tecnologías utilizadas

- Python 3
- FastAPI
- Uvicorn
- Redis
- Docker
- Docker Compose
